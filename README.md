# create-liminalis-app

Scaffold a new Liminalis music visualization project with zero configuration.

## Usage

```bash
npm create liminalis-app my-music-viz
cd my-music-viz
npm install
npm run dev
```

Or with other package managers:

```bash
# yarn
yarn create liminalis-app my-music-viz

# pnpm
pnpm create liminalis-app my-music-viz

# npx
npx create-liminalis-app my-music-viz
```

## What's Included

Each template includes:

- ✅ TypeScript support
- ✅ Vite for fast development and HMR
- ✅ Liminalis framework pre-configured
- ✅ Example visualizations
- ✅ Production build setup

## Requirements

- Node.js 16+
- A modern browser (Chrome, Edge, Opera for WebMIDI support)
